# HHApp
NodeJS and Angular 2 Shopping App

### Prerequisites

```
Node
Nodemon (optional)
Angluar Cli

```

### Installing

Back-End commands to running project

cd to the project HHApp

```
npm install

node app

```

Front-End commands to running project

cd to the project \HHApp\shoppingwebapp

```
npm install

ng serve

```
